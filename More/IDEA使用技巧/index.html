# 1.Module

一个IDEA项目对应一个Project，一个Project对应多个Mudule 。 可以 Command + ； 唤出项目结构新建或解除Mudule

且只有解除的Module才能删除（ps:可以防止误删）

# 2. 新UI（需2022.3版本）

  1. 随便进入一个项目
  2. 双击 shift，搜索registry
  3. 将ide.experimental.ui，勾上
  4. 重启即可（推荐Dark主题）
